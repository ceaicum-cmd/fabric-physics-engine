# Fabric Physics Prompt Engine v3 (2026)

## Purpose
The Fabric Physics Prompt Engine is a modular prompting framework designed to improve clothing realism in AI-generated and AI-edited imagery.

The framework forces the model to simulate believable fabric behavior through:
- gravity
- tension
- compression
- material stiffness
- weight distribution
- seam behavior
- fold hierarchy
- anatomical interaction
- layered garment structure

Optimized for:
- body reshaping
- fashion photography prompts
- character redesigns
- stylized realism
- high-detail portrait generation
- clothing correction after anatomy edits
- preventing painted-on clothing artifacts
- AI inpainting and image editing workflows

## Core Engine Principle
**Central rule:** body modifications must propagate naturally into fabric behavior through gravity, tension, compression, draping, seam interaction, and material response.

Any anatomical adjustment should physically influence:
- fold placement
- tension direction
- fabric stretching
- compression density
- garment volume
- hanging weight
- seam stress
- wrinkle hierarchy

## Model-Specific Optimization
- **Flux / Flux Kontext:** repeat gravity and physical behavior concepts at least twice.
- **SDXL:** use fold generation and seam logic aggressively.
- **Midjourney:** prioritize cinematic phrasing and concise visual descriptors.
- **Realistic Vision / Photoreal:** emphasize compression, tension, gravity, and micro-wrinkle detail.

## v3 System Fixes (Failure-Mode Driven)
Targets:
- uniform smoothness collapse
- skin-fused garments
- missing force propagation
- unrealistic compression behavior
- absent material identity
- symmetric wrinkle artifacts
- non-physical stretch distribution

Adds:
- anisotropic stretch simulation
- contact deformation logic
- tension decay systems
- micro-asymmetry generation
- force-chain propagation
- volumetric thickness preservation
- advanced compression instability modeling

## Engine Summary
The framework improves realism by forcing the model to treat clothing as a physical system rather than a surface texture.
