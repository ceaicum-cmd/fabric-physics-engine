# Fabric Realism Scorecard

## How to Use
Score each criterion from **0 to 2**:
- **0:** clearly broken / artifact-heavy
- **1:** partially correct but inconsistent
- **2:** physically believable and consistent

Total possible score: **20**.

## Criteria
1. **Gravity drape realism** (downward pull, weighted hanging)
2. **Tension logic** (anchor-based stress lines and transitions)
3. **Compression behavior** (organic bunching and wrinkle density)
4. **Fold hierarchy** (primary folds + secondary micro-structure)
5. **Material identity** (behavior matches fabric type)
6. **Seam construction realism** (seams guide tension/folds)
7. **Layer interaction** (overlap, spacing, displacement)
8. **Thickness/volume preservation** (no skin-fused look)
9. **Micro-detail quality** (subtle wrinkles, edge tension, seam puckering)
10. **Artifact resistance** (no plastic/pasted-on/melted texture cues)

## Score Bands
- **17–20:** Production-ready realism
- **13–16:** Good realism with targeted refinements needed
- **9–12:** Moderate artifact risk; escalate physics modules
- **0–8:** Severe failure; rebuild prompt with L3 stack

## Escalation Rules
If any critical criterion is **0** (thickness, tension, compression, fold hierarchy, anti-artifact), do all:
1. Move up one intensity level
2. Add explicit seam and force-propagation language
3. Reassert material-specific behavior
4. Re-run generation with same seed framing if supported
